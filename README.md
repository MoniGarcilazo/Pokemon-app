# PokéMoni 🔍⚡

**PokéMoni** es una pequeña aplicación tipo **Pokédex** desarrollada con **React, TypeScript, Redux Toolkit y Vite**.
Permite explorar Pokémon, buscarlos por nombre y ver sus detalles en un modal.

La aplicación utiliza la **PokéAPI** para obtener la información de los Pokémon e implementa funcionalidades como **búsqueda con autocompletado y paginación**.

---

## ✨ Funcionalidades

* 🔎 Búsqueda de Pokémon por **nombre**
* ⚡ **Autocompletado** mientras escribes
* 📄 **Paginación** para navegar entre Pokémon
* 🧾 **Modal con detalles** del Pokémon seleccionado
* 🎨 Interfaz con temática de Pokémon
* ⚙️ Manejo de estado global con **Redux Toolkit**

---

## 🧰 Tecnologías utilizadas

* **React**
* **TypeScript**
* **Redux Toolkit**
* **Vite**
* **TailwindCSS**
* **PokéAPI**

---

## 📦 Instalación

Clona el repositorio:

```bash
git clone https://github.com/tu-usuario/pokemoni.git
```

Entra a la carpeta del proyecto:

```bash
cd pokemoni
```

Instala las dependencias:

```bash
npm install
```

---

## ▶️ Ejecutar el proyecto en local

Inicia el servidor de desarrollo:

```bash
npm run dev
```

La aplicación estará disponible en:

```
http://localhost:5173
```

---

## 🌐 API

Este proyecto utiliza la **PokéAPI**, una API pública con información de Pokémon:

https://pokeapi.co/

No requiere API key.
